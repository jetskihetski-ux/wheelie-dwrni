# Wheelie DWRNI

Autonomous mobile vending robot — INJAZ UAE Company Program 2019–2020.

Wheelie DWRNI navigates a space on wheels and dispenses on-demand supplies (medicine, hygiene products, snacks) to customers without them needing to go to a fixed vending machine.

## Features
- **Inventory system** — product database with stock tracking, categories, and sales reporting
- **Robot navigation** — grid-based movement with real-time position, heading, and battery monitoring
- **Obstacle detection** — ultrasonic sensor simulation with automatic rerouting
- **Order & dispense** — customer order flow with transaction logging
- **Path planning** — BFS shortest-path algorithm for efficient routing
- **Low stock alerts** — automatic warnings when supplies run low
